# 4. 인터페이스 정의

## 4.1 인터페이스 유형 개요

| 유형 | 프로토콜 | 용도 | 관련 모듈 |
|------|----------|------|-----------|
| REST API | HTTP/JSON | 대화/첨부/지식 CRUD, 모델 조회 | conversations, attachments, knowledge, models |
| SSE 스트리밍 | HTTP/SSE | 채팅 응답 토큰 스트리밍 | chat |
| 외부 HTTP | HTTP/JSON | Ollama LLM API 호출 | chat, knowledge, models |
| 파일 업로드 | HTTP/multipart | 첨부파일, 지식문서 업로드 | attachments, knowledge |
| 파일 다운로드 | HTTP/octet-stream | 대화 내보내기 | conversations |

---

## 4.2 REST API — 대화 관리 (`/api/conversations`)

| Method | Path | 설명 | 요청 | 응답 |
|--------|------|------|------|------|
| GET | `/api/conversations` | 대화 목록 조회 (updated_at DESC) | - | `ConversationResponse[]` |
| GET | `/api/conversations/search?q={keyword}` | 대화/메시지 검색 (LIKE) | Query: `q` (min 1자) | `SearchResult[]` (최대 50건) |
| POST | `/api/conversations` | 대화 생성 | `ConversationCreate` | `ConversationResponse` (201) |
| GET | `/api/conversations/{id}` | 대화 상세 (메시지 포함) | Path: `id` | `ConversationDetailResponse` |
| PATCH | `/api/conversations/{id}` | 대화 수정 | `ConversationUpdate` | `ConversationResponse` |
| DELETE | `/api/conversations/{id}` | 대화 삭제 (CASCADE) | Path: `id` | 204 No Content |
| GET | `/api/conversations/{id}/export?format={fmt}` | 대화 내보내기 | Query: `format` (json\|markdown) | 파일 다운로드 |
| POST | `/api/conversations/import` | 대화 가져오기 | `ConversationImport` | `ConversationResponse` (201) |

### 요청/응답 스키마

#### ConversationCreate

```json
{
  "title": "새 대화",
  "model": "gemma4:26b",
  "system_prompt": "너는 전문 번역가야..."  // nullable
}
```

#### ConversationUpdate

```json
{
  "title": "수정된 제목",        // nullable
  "model": "llama3:8b",         // nullable
  "system_prompt": "프롬프트..."  // nullable, 빈 문자열 → null
}
```

  "updated_at": "2026-05-07T10:30:00"

## 4.3 SSE 스트리밍 — 채팅 (`/api/conversations/{id}/chat`)

| Method | Path | 설명 |
|--------|------|------|
| POST | `/api/conversations/{id}/chat` | SSE 스트리밍 채팅 |

### 요청

```json
{
  "message": "사용자 메시지 내용"
}
```

### SSE 이벤트 형식

모든 이벤트는 `data: {JSON}\n\n` 형식으로 전송됩니다.

| 이벤트 타입 | JSON 구조 | 설명 |
|-------------|-----------|------|
| thinking | `{"thinking": "토큰"}` | 모델 사고 과정 토큰 (thinking 모델만) |
| token | `{"token": "토큰"}` | 응답 본문 토큰 |
| done | `{"done": true, "title": "제목", "references": [...]}` | 응답 완료 (title은 첫 메시지만) |
| error | `{"error": "에러 메시지"}` | 오류 발생 |

### references 배열 구조

```json
[
  {
    "filename": "문서명.pdf",
    "score": 0.875,
    "matched_summary": true
  }
]
```

| 필드 | 타입 | 설명 |
|------|------|------|
| filename | string | 참조된 지식 문서 파일명 |
| score | float | BM25+TF-IDF 하이브리드 매칭 점수 (0~1) |
| matched_summary | boolean | 문서 요약 매칭 여부 (우선 문서) |

## 4.4 REST API — 첨부파일 (`/api/conversations/{id}/attachments`)

| Method | Path | 설명 | 요청 | 응답 |
|--------|------|------|------|------|
| POST | `/{conv_id}/attachments` | 파일 업로드 (최대 10MB) | multipart/form-data (`file`) | 첨부파일 정보 JSON |
| GET | `/{conv_id}/attachments` | 첨부파일 목록 조회 | - | 첨부파일 배열 |
| DELETE | `/{conv_id}/attachments/{att_id}` | 첨부파일 삭제 | - | 204 No Content |

### 첨부파일 응답

```json
{
  "id": 1,
  "filename": "문서.pdf",
  "file_size": 102400,
  "created_at": "2026-05-07T10:00:00"
}
```

## 4.5 REST API — 지식 저장소 (`/api/knowledge`)

| Method | Path | 설명 | 요청 | 응답 |
|--------|------|------|------|------|
| GET | `/api/knowledge` | 지식 문서 목록 | - | 문서 배열 |
| POST | `/api/knowledge/upload` | 지식 문서 업로드 (최대 20MB) | multipart/form-data (`file`) | 문서 정보 JSON |
| GET | `/api/knowledge/{id}/status` | 문서 처리 상태 조회 | Path: `id` | 상태 JSON |
| DELETE | `/api/knowledge/{id}` | 지식 문서 삭제 | Path: `id` | 204 No Content |

### 문서 목록 응답

```json
{
  "id": 1,
  "filename": "spec.pdf",
  "file_size": 5242880,
  "chunk_count": 42,
  "summary": "이 문서는 5G NR 표준에 대한...",
  "status": "ready",
  "error_message": null,
  "created_at": "2026-05-07T10:00:00"
}
```

### 문서 상태

| status | 설명 |
|--------|------|
| `processing` | 텍스트 추출/요약 생성/청크 분할 진행 중 |
| `ready` | 처리 완료, 검색 가능 |
| `error` | 처리 실패 (error_message에 상세 내용) |

## 4.8 외부 연동 — Ollama API

프론트엔드 → 백엔드 → Ollama 순으로 프록시하여 호출합니다.

| Ollama API | 백엔드 호출 위치 | Method | 용도 | Timeout |
|------------|-----------------|--------|------|---------|
| `/api/chat` | chat.py::generate() | POST (stream) | 대화 스트리밍 응답 | 300s |
| `/api/generate` | chat.py::generate_title() | POST | 자동 제목 생성 | 60s |
| `/api/generate` | knowledge.py::generate_summary() | POST | 문서 요약 생성 | 120s |
| `/api/tags` | models.py::list_models() | GET | 모델 목록 조회 | 10s |
